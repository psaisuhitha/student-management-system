# 🎓 Student Management System

A Full Stack Web Application built with React, Node.js, Express and MongoDB.

## 🚀 Features
- View all students
- Add new student
- Edit student details
- Delete student
- Search by name

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Tools:** Mongoose, Axios, CORS

## ⚙️ Installation

### Backend:
cd backend
npm install
npm start

### Frontend:
cd frontend
npm install
npm start
